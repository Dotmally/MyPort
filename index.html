Work

Pit Escape Simulation.

This project features a C++ program that simulates the escape of Timmy Tortoise and Prince Harold the Hare from the dangerous Snorc Pit, with help from stealth Ninjas against snake orcs (snorcs). Using the Strategy design pattern and dynamic polymorphism, the simulation displays dynamic interactions and combat scenarios. Each character—from heroes to ninjas and snorcs—has unique behaviors influenced by random events, showcasing my ability to develop engaging and complex software solutions.

Ghost Hunt Simulator.

This project is a multi-threaded simulation that models a ghost hunt in a dynamic environment. Using systems programming techniques, I designed interactions between hunters and a ghost within a haunted house. The simulation manages entity behaviors such as movement and evidence collection, utilizing threads for real-time actions and semaphores to prevent deadlocks. This project demonstrates my capability in applying advanced programming concepts to create complex, interactive simulations.
